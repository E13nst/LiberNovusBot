# Capability Milestone: Prompt Compiler

## Outcome

Prompt compiler foundation is implemented for controlled Jungian analysis prompting.

## Includes

- prompt builder composition from dreams and summaries;
- prompt contract definition;
- prompt validation layer;
- anti-hallucination anchors in compiled prompts.

## Invariants

- prompt assembly is deterministic;
- prompts remain inspectable and testable;
- provider integration stays isolated from core prompt logic.
