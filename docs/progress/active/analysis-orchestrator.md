# Active Capability: Analysis Orchestrator

## Goal

Build a multi-stage analysis orchestrator that transforms compiled session context into validated structured analysis outputs.

## Why now

Prompt compiler and deterministic layers are already in place. The next architectural milestone is controlled reasoning orchestration, not UI expansion.

## Scope

- stage orchestration contract;
- motif extraction stage;
- emotional field stage;
- archetypal hypotheses stage;
- compensation analysis stage;
- question generation stage;
- structured output validation.

## Out of scope

- frontend visualization;
- embeddings / vector DB;
- archetype graph engine;
- provider-specific business logic.

## Done criteria

- orchestration flow is test-covered;
- every stage has explicit input/output contract;
- outputs are JSON-first and schema-validated;
- provider adapters remain isolated.
