# Видение продукта

Liber Novus не является ботом для толкования снов.

Проект задуман как рефлексивный спутник: система, которая помогает человеку бережно собирать сновидения, видеть повторяющиеся символические мотивы и возвращаться к ним в долгом контексте. Цель не в том, чтобы выдать готовое значение сна, а в том, чтобы поддержать исследовательский диалог с собственным материалом.

## Продуктовая позиция

Liber Novus должен оставаться:

- детерминированным там, где это возможно;
- объяснимым и проверяемым;
- независимым от конкретного AI-провайдера;
- психологически осторожным;
- ориентированным на долгосрочную историю пользователя.

Система не подменяет терапию, диагностику или авторитетное психологическое заключение. Она помогает структурировать материал, задавать вопросы и замечать связи между снами, эмоциями, образами и жизненным контекстом.

## Не "dream interpretation bot"

Антипаттерн продукта:

```text
сон -> промпт -> готовая интерпретация
```

Желаемая модель:

```text
сны
  -> сессии
  -> детерминированная структура
  -> осторожные аналитические гипотезы
  -> вопросы для размышления
  -> накопление контекста во времени
```

## Эволюционный путь

Развитие продукта должно идти от надежного хранилища к системе символического анализа:

```text
журнал сновидений
    ↓
сессионный контекст
    ↓
структурированные summary и мотивы
    ↓
reasoning layer
    ↓
рефлексивный companion
```

Приоритет проекта на текущем этапе: построить foundation для reasoning layer, а не добавлять преждевременные визуализации, графы, embedding search или сложный UI.
