# Future Roadmap

Roadmap должен развивать reasoning layer постепенно. Нельзя перескакивать к визуализациям, embeddings или графам до того, как стабилизированы данные, сессии, prompt contract и structured outputs.

## Near-term

- Analysis Orchestrator.
- Provider abstraction.
- JSON schema validation для AI-ответов.
- Persistence для structured analysis.
- Question Generator.
- Tests для prompt compilation и validation.

## Mid-term

- Archetype evolution engine.
- Symbolic graph engine.
- Emotional recurrence tracker.
- Compensation analysis engine.
- Session-level semantic memory.

## Long-term

- Hybrid local/cloud inference.
- Local models для приватных сценариев.
- Semantic embedding search.
- Analyst collaboration mode.
- Group symbolic field analysis.
- Экспорт исследовательского архива пользователя.

## Принцип приоритизации

Каждый новый слой должен отвечать на вопрос: что он делает более объяснимым, проверяемым или полезным для долгосрочного исследования материала?

Если слой увеличивает впечатляющую сложность, но не повышает explainability, его нужно отложить.
