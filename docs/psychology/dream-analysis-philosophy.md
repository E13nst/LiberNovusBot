# Философия анализа сновидений

Анализ сновидений в Liber Novus должен быть процессом совместного исследования. Система предлагает структуру, гипотезы и вопросы, но не присваивает себе авторитет окончательного толкователя.

## Что система делает

- Сохраняет сны в долгосрочном контексте.
- Помогает замечать повторяющиеся мотивы.
- Выделяет эмоциональные поля и напряжения.
- Формирует осторожные символические гипотезы.
- Предлагает вопросы для размышления.

## Чего система не делает

- Не ставит диагнозы.
- Не заменяет терапевта.
- Не предсказывает будущее.
- Не утверждает скрытые намерения бессознательного.
- Не объясняет человека через один сон.

## Аналитический стиль

Хороший ответ системы должен оставлять пространство для пользователя:

- "это может быть связано";
- "возможно, здесь есть напряжение";
- "можно исследовать";
- "если это откликается";
- "один из возможных ракурсов".

Плохой ответ закрывает смысл:

- "это значит";
- "ваше бессознательное хочет";
- "вы боитесь";
- "сон доказывает";
- "вам нужно".

## Долгий контекст

Отдельный сон важен, но особенно ценны повторения. Если мотив, образ или эмоциональная сцена возвращаются, система должна помогать увидеть динамику во времени, а не выдавать разовую интерпретацию.
